Department

Only admin can access department module. The system does not allow disable a department, has an active employee.


Create Department
1. Create new department navigate to 'New' from the department list.

1.a Admin clicks on 'Department' -> 'New'
1.b Admin types department name, ‘Quality Control’, clicks on ‘Add’, new department, ‘Quality Control’, added successfully.

View User

2. View department details with ‘Back’ button

2.a Admin views the department list
2.b Clicks on ‘Action’ of ‘Quality Control’ changed the department name to ‘Quality Assurance’,

clicks on ‘Save’, department Name changed to ‘Quality Assurance’ from ‘Quality Control’
2.c Admin types ‘Finance’, clicks on search, the system display department list of ‘Finance’
2.d Admin clicks on ‘Finance’ to view employee list of the department.
2.e.i 'Finance' department has the active employees, hence admin does allow to change department status

'True' to 'False', the system will display an error message.
2.e.ii Admin change status ‘False’ or ‘Inactive’ of ’Quality Assurance’
2.f. Clicks on 'map' icon, the system moves to map between the department and

part/ document, for details please go through on ‘Mapping’ module

Users

1. Create new user navigate to 'New' from the user list.
1.a Admin clicks on 'User' -> 'New'
1.b The mandatory fields are first name, last name, designation, email, password, mobile, hire date, and department(s).
1.c Admin enters data on mandatory fields.
1.d Admin can add a new department by clicks on '++', type department name, press enter key.
1.e Admin allocates 'Quality Control', 'Administration', 'Manufacturing', and 'Finance' to David
1.f Clicks on remove, 'Finance' wipes out from allocated user department list.
1.g Clicks on 'Save'
2.) View user details with ‘Notify’ and ‘Edit’
2.a clicks on ‘Notify’ .
2.b success message ‘Notify’.
2.c clicks on ‘Edit’.
3.) Edit user details with ‘Save’ and ‘Back’
3.a change mobile # and clicks on save.
3.b view user details.
3.c reset password and clicks on save.
3.d view user details.
3.d system does not allow to remove all department of an user
4.) Admin views ‘User list’
4.a user list.
4.b Types ‘John’, clicks on search and sort with active status.

User

The module is only available to document control admin (DCA).



Create User
1.a The user navigates on 'User' -> 'New' to create a user, from the list.

1.b The user enters data all mandatory fields.
1.c The mandatory fields are first name, last name, designation, email, password, mobile, hire date, and department(s).
1.d The user(admin) allows to create an admin ( 'Is Admin' = 'Yes').
1.e The user can create a new department by clicks on ‘++’, type department name, press enter key.

1.f The users allocate department ‘Quality Assurance’, ‘Operations’, and 'Finance' to David Villa.
1.g The system removes, 'Finance' wipes out from department list.
1.h The user clicks on 'Save'. The system creates the new user, and displays on the user list.

View User

2.a The user clicks on username 'David Villa' from the list, the system moves to user details page ('David Villa').

2.b Admin clicks on ‘Notify User’, the system send an email notification to 'David Villa'.
2.c ‘Back’ moves to the user list.

Edit User

3.a The user clicks on ‘Edit’, the system displays edit user details of ‘David Villa’.


3.b The user clicks on 'Reset Password', the system displays automated password.

3.c The user changes ‘Mobile no’ to ‘9874563212’ from ‘9874563210’, and clicks on ‘Save’, the user, David's mobile number updates successfully.

List of Users

4. The user views ‘User list’.
4.a The user can set status 'active' to 'inactive' or vice versa, The default is active.
4.b The user types 'John', clicks on search, the system displays result.

4.c The user clicks on sort button of name, the system displays sorted list. The system also allow sort on 'Email', 'Status', and 'Is active' fields.

Users

The ‘User’ module allows to creates a new, edit, view user, and a list, the module is available only to DCA (Document Control Admin).
1. Create new user navigate to 'New' from the user list.
1.a Admin clicks on 'User' -> 'New'
1.b The mandatory fields are first name, last name, designation, email, password, mobile, hire date, and department(s).

1.c Admin enters data on mandatory fields.

1.d Admin can add a new department by clicks on '++', type department name, press enter key.
1.e Admin allocates 'Quality Assurance', 'Administration', 'Manufacturing', and 'Finance' to David
1.f Clicks on remove, 'Finance' wipes out from allocated user department list.
1.g Admin clicks on 'Save'. The user 'David Villa' saved, and display on the user list.
2. View user details with ‘Notify’, ‘Edit’, and ‘Back’
2.a Admin clicks on ‘Notify’, the system sent an email notification to 'David' .
2.b Admin clicks on 'Edit', the system opens to edit user details of 'David'. 2.c ‘Back’ moves to the user list
3. Edit user details with 'Save', 'Reset Password', 'Cancel'
3.a The system displayed automated password when admin clicks on 'Reset password'.
3.b Admin changed ‘Mobile No’ to ‘9898252598’ from ‘9898252525’, and clicked on ‘Save’, the user, David's mobile number updated successfully.
3.c reset password and clicks on save.
4. Admin views ‘User list’
4.a Admin can set status 'active' to 'inactive' or vice versa. The default state is active.
4.b Admin types 'John', clicks on search, the system displayed searched list.
4.c Admin clicks on sort button user name, the system displayed sorted list. The system also allow sort on 'Email',

'Contact no', 'Status', 'Is active'.
